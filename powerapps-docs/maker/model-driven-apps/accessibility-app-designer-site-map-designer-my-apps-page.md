# <a name="accessibility-in-powerapps-app-designer-site-map-designer-and-my-apps-page"></a>Barrierefreiheit im PowerApps-App-Designer, Siteübersichts-Designer, Seite „Meine Apps“

Microsoft ist bestrebt, seine Produkte und Dienste für alle Benutzer zu vereinfachen. Weitere Informationen finden Sie unter [Barrierefreiheit bei Microsoft](http://www.microsoft.com/enable/default.aspx).

## <a name="keyboard-navigation"></a>Tastaturnavigation  
 Darüber hinaus wird das Springen innerhalb des App-Designers, des Siteübersichts-Designers und der Seite „Meine Apps“ mithilfe der Tastatur unterstützt, sodass Personen, die keine Maus verwenden, eine Tastatur verwenden können, um Aktionen auf eine andere Art und Weise durchzuführen.  
  
|Bis|Tastenkombination|  
|--------|-----------|  
|Navigieren zum nächsten Tabstopp in der definierten Aktivierreihenfolge (d.h. von links nach rechts bei LTR-Sprachen und von rechts nach links bei RTL-Sprachen).|TAB|  
|Springen zum vorhergehenden Tabstopp in der Aktivierreihenfolge|UMSCHALT+TAB|  
|Springen zum nächsten Element in einer Liste oder einem Menü oder zum Feature der Benutzeroberfläche, das über mehrere Optionen verfügt<br /><br /> ODER<br /><br /> Vertikales Springen zur nächsten Kachel|NACH-UNTEN-TASTE|  
|Springen zum nächsten Element in einer Liste oder einem Menü oder zum Feature der Benutzeroberfläche, das über mehrere Optionen verfügt<br /><br /> ODER<br /><br /> Vertikales Springen zur vorherigen Kachel|NACH-OBEN-TASTE|  
|Horizontales Springen zur nächsten Kachel|NACH-RECHTS-TASTE|  
|Horizontales Springen zur vorherigen Kachel|NACH-LINKS-TASTE|  
|Erweitern eines reduzierbaren Steuerelements|NACH-RECHTS-TASTE, EINGABETASTE oder LEERTASTE|  
|Reduzieren eines erweiterten reduzierbaren Steuerelements|NACH-LINKS-TASTE, EINGABETASTE oder LEERTASTE|  
|Ausführen einer Aktion wie das Aktivieren einer Option über Optionsfelder, das Klicken auf einen Link oder das Aktivieren eines Kontrollkästchens|LEERTASTE|  
|Aktivieren einer Befehlsleistenaktion oder Bestätigen einer hinzuzufügenden Komponente|EINGABETASTE|  
|Schließen einer Warnung, eines Dialogfelds oder eines Menüs oder Abbrechen dergleichen|ESC-TASTE|  
  
## <a name="screen-reader-support"></a>Unterstützung der Sprachausgabe  
 Personen mit Behinderungen können auf Hilfstechnologien wie die Sprachausgabe oder eine Vielzahl von alternativen Eingabegeräten zurückgreifen, um mit dem App-Designer, dem Siteübersichts-Designer und der Seite „Meine Apps“ zu interagieren.  
  
 Im Folgenden finden Sie die Supportmatrix.  
  
|Browser|Jaws|Sprachausgabe|VoIP|  
|-------------|----------|--------------|----------------|  
|Internet Explorer 11 |Ja|Ja|Nein|  
|Google Chrome |Ja|Nein|Nein|  
| Firefox |Ja|Nein|Nein|  
|Safari|Nein|Nein|Ja|
