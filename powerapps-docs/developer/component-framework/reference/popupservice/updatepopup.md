## <a name="available-for"></a>Verfügbar für

## <a name="syntax"></a>Syntax

`updatePopup(name, newProps)`

## <a name="parameters"></a>Metern

| Parameter Name|Typ|Erforderlich|Beschreibung|
| ------------- |----|--------|-----------|
|Benennen|`String`|Ja|Der Name des Popups, das Sie aktualisieren möchten.|
|neueigenschaften|[Popup](../popup.md)|Nein|Die aktualisierten Eigenschaften, die dem Popup zugewiesen werden sollen.|
