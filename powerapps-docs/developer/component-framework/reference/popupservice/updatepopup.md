# <a name="updatepopup"></a>updatePopup

[!INCLUDE [updatepopup-description](includes/updatepopup-description.md)]

## <a name="syntax"></a>Syntax

`updatePopup(name, newProps)`

## <a name="parameters"></a>Parameter

| Parametername|Typ|Erforderlich|Beschreibung|
| ------------- |----|--------|-----------|
|Name|`string`|Ja|Der Name des Popups, den Sie zu aktualisieren versuchen.|
|newProps|`Popup`|Nein|Die aktualisierten Eigenschaften, die dem Popup übergeben werden sollen.|
