# <a name="subgrid-onload-event-client-api-reference"></a>Unterraster OnLoad-Ereignis (Client-API-Referenz)



Dieses Ereignis tritt auf, wenn das Unterraster aktualisiert wird. Dies gilt auch dann, wenn Benutzer Werte im Unterraster sortieren, indem sie auf die Spaltenüberschriften klicken. 

Verwenden Sie die GridControl.addOnLoad- und GridControl.removeOnLoad-Methoden, um Ereignishandler zu verwalten, normalerweise im Formular Onload-Ereignis.
