## <a name="syntax"></a>Syntax

`formContext.ui.process.getVisible();`

## <a name="return-value"></a>Rückgabewert

**Typ:**: Boolesch.

**Beschreibung**: „True“, wenn das Steuerelement sichtbar ist, andernfalls „false“.
