# <a name="formcontextdataentity-client-api-reference"></a>formContext.data.entity (Client-API-Referenz)



Stellt Eigenschaften und Methoden zum Abrufen von Informationen bereit, die für den auf der Seite angezeigten Datensatz spezifisch sind, die Speichermethode und eine Sammlung aller im Formular enthaltenen Attribute. Attribute werden durch Felder beschränkt, die im Formular angezeigt werden.

## <a name="properties"></a>Eigenschaften

|Name|Beschreibung|
|--|--|
|Attribute|Sammlung von Attributen für einen Datensatz, der angezeigt wird im Formular. <br/>Weitere Informationen: [Sammlungen](collections.md) und [Attribute](attributes.md).
