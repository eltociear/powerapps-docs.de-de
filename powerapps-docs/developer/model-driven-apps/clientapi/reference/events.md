# <a name="events-client-api-reference"></a>Ereignissse (Client-API-Referenz)



Ereignisse treten auf in benutzerdefinierten Engagementformularen und -Rastern, wenn ein Formular oder ein Raster geladen wird, Daten gespeichert oder geändert werden. Sie führen den JavaScript-Code aus, indem Sie ihn Ereignissen zuordnen, damit er ausgeführt werden kann, wenn das Ereignis eintritt. Weitere Informationen: [Ereignisse in Formularen und in Rastern](../events-forms-grids.md)
